# Daily Routines

A simple mobile app (MVP) for building daily routines, getting reminders to complete
them, and tracking completion over time. Built with React + Vite and packaged for
Android using [Capacitor](https://capacitorjs.com/).

## Features

- **Routines** — create routines with a name, reminder time, repeat days, and optional notes.
- **Reminders** — local push notifications on your Android device at the routine's scheduled time/days.
- **Today** — a daily checklist of everything due today, with one tap to mark complete.
- **History** — a 14-day grid per routine plus current streak and 30-day completion rate.

Data is stored on-device (via Capacitor Preferences) — no backend/server required.

## Project structure

- `src/storage.js` — persistence layer (routines + completion history).
- `src/notifications.js` — schedules/cancels local notifications per routine.
- `src/utils/date.js` — date helpers, streak and completion-rate calculations.
- `src/components/` — `RoutineForm`, `RoutinesView`, `TodayView`, `HistoryView`.
- `android/` — native Android project generated by Capacitor.

## Run in the browser (fastest way to iterate on the UI)

```bash
npm install
npm run dev
```

Note: local notifications only fire on a native Android build — in the browser the
app runs fine but reminders are skipped.

## Build and run on Android

You'll need [Android Studio](https://developer.android.com/studio) installed locally
(this dev environment doesn't have the Android SDK, so the APK must be built on your
machine).

1. Build the web app and sync it into the Android project:
   ```bash
   npm install
   npm run build
   npx cap sync android
   ```
2. Open the native project in Android Studio:
   ```bash
   npx cap open android
   ```
3. In Android Studio, connect a device (USB debugging enabled) or start an emulator,
   then click **Run**. Android Studio handles signing/building the debug APK for you.

Whenever you change the web code, re-run `npm run build && npx cap sync android`
before rebuilding in Android Studio.

## App id

The Android app id is `com.tharuka.routines` (set in `capacitor.config.json`) —
change it there (and in `android/app/build.gradle`) before publishing.
